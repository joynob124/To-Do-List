## ToDo 

A very simple To-Do List application built with **Java Swing**.

## Features
- Add, delete, and delete all tasks
- Tasks save automatically
- Inlcude multiple tasks

## How to Run
- Download the `ToDo.jar` file
- Open terminal and 
- Run: `java -jar ToDo.jar` 
  .... and enjoy how it works!
